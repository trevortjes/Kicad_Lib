<p></p>
1) Download trevotjes_lib.pretty and place it somewhere you can easily find it
<p></p>
2) In KiCad, Preferences > Configure Paths 
<p></p>
Add a new entry with Name "KICAD_USER_3D" and as path "trevortjes.lib.pretty\3D"
<p></p>
3) Now go to Preferences > Manage Footprint Libraries
<p></p>
Add a new entry with Nickname "trevortjes_lib" and library path to the trevortjes_lib.pretty folder
<p></p><p></p>
Done!
